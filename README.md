##Install
```
go get github.com/kevinbin/mysqlmon   
```
## Usage
```
Welcome to the MySQL Toolbox. 

Version: 1.0

Usage:
  mysqlmon [command]

Available Commands:
  help        Help about any command
  monitor     A MySQL monitor like iostat

Flags:
  -h, --help              help for mysqlbox
  -H, --host string       mysql host ip (default "localhost")
  -p, --password string   mysql login password (default "root")
  -P, --port int          mysql server port (default 3306)
  -u, --user string       mysql login user (default "root")

Use "mysqlbox [command] --help" for more information about a command.
```
